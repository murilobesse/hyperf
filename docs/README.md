## 🚀 Início Rápido

Para começar a usar o projeto:

1. **Suba o container:**
   ```bash
   docker-compose up -d
   ```

2. **Acesse a API:**
   ```
   http://localhost:9501
   ```

3. **Consulte a documentação:**
   - [Estrutura do Projeto](estrutura.md) - Entenda a organização
   - [Docker](docker.md) - Comandos úteis
   - [Watcher](watcher.md) - Hot reload automático
